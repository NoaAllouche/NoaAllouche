## 🎮 Text Adventure World [git link](https://github.com/NoaAllouche/Text-Adventure-World.git)

A 2-player cooperative console adventure game developed in C++.

- Supports file-based room loading and interactive gameplay mechanics.

---

## ♞ CheckMate Project [git link](https://github.com/NoaAllouche/CheckMate.git)

A chess knight path-finding project developed in C.

- Implements recursive tree traversal, linked lists, dynamic memory allocation, and backtracking algorithms.
- Generates all valid knight moves and searches for a path covering the entire chess board.
- Includes visualization of knight traversal paths.

---

## 🐂 Bulls and Cows Solver [git link](https://github.com/NoaAllouche/Bulls-And-Cows-Solver.git)

A Python implementation of the classic Bulls and Cows game solver.

- Generates valid unique-digit codes and evaluates Bulls/Cows feedback.
- Dynamically filters candidates after each guess.
- Includes multiple solving strategies, including a Knuth-based optimal guessing algorithm.
